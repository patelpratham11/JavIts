# JaVits
    A Java-based gamification of your life

## Usage
**Preconditions**
- Must have Java Installed
- Clone this repository onto your local machine
- Set up the `.config` files within the `Resources` folder

**Execution**
- Navigate to the correct directory
- `java -jar JaVits.jar`
